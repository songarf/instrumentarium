## 1. Начало работы

### Скачивание

1. Перейди на [страницу релизов](https://github.com/songarf/instrumentarium/releases)
2. Скачай архив для своей системы:
   - **Windows** → `Instrumentarium-windows.zip`
   - **Linux** → `Instrumentarium-linux.tar.gz`
   - **macOS** → `Instrumentarium-macos.tar.gz`

### Запуск

1. Распакуй архив в любую папку
2. Запусти `Instrumentarium.exe` (Windows) или `Instrumentarium` (Linux/macOS)

> **Важно:** Не удаляй файлы рядом с .exe — всё приложение хранится в одной папке.

## 2. Первый запуск

При первом запуске приложение автоматически установит необходимые компоненты:

1. **Проверка Python** — ищет Python 3.7+ в системе. Если не найден — установит автоматически (Windows) или покажет инструкцию (Linux/macOS)
2. **Установка yt-dlp** — скачивается в папку `.bin/`
3. **Установка ffmpeg** — скачивается автоматически на Windows. Нужен для объединения видео и аудио

Прогресс установки отображается в окне. Обычно занимает 1-3 минуты в зависимости от скорости интернета.

> **Примечание:** При повторном запуске настройка пропускается — приложение сразу готово к работе.

## 3. Cookies (для приватных страниц)

Некоторые платформы (LinkedIn и др.) требуют авторизацию для доступа к видео. Для этого нужен файл cookies из вашего браузера.

### Как получить cookies.txt

1. Установите расширение **"Get cookies.txt LOCALLY"** в Chrome/Edge
2. Откройте linkedin.com и войдите в аккаунт
3. Нажмите на иконку расширения → **Export**
4. Файл cookies.txt сохранится на ваш компьютер

### Как загрузить в приложение

1. Нажмите **🍪 Cookies** внизу окна
2. Перетащите файл cookies.txt в окно (drag & drop) или вставьте текст вручную
3. Нажмите **Сохранить** → окно закроется автоматически

> Подсказку (?) о том как получить cookies можно найти в диалоге cookies.

## 4. Скачивание видео

### Шаг 1: Вставь ссылку

1. Скопируй ссылку на видео из браузера
2. Вставь в поле ввода в верхней части окна
3. Приложение автоматически определит платформу и покажет бейдж (🔴 YouTube, 📸 Instagram и т.д.)

### Шаг 2: Дождитесь анализа

После вставки ссылки приложение проанализирует видео (2-5 секунд). Появятся кнопки с доступными разрешениями.

### Шаг 3: Выбери качество

Нажми на кнопку с нужным разрешением:

| Кнопка | Описание |
|--------|----------|
| **1080p** | Полное HD качество |
| **720p** | HD качество (меньше файл) |
| **480p** | Среднее качество |
| **Скачать видео** | Единственный доступный формат (для некоторых платформ) |

Справа на кнопке показан размер файла (например, "2.9 MB · MP4").

### Шаг 4: Дождитесь загрузки

Прогресс-бар покажет ход загрузки. После завершения:
- Прогресс-бар станет зелёным
- Статус покажет "✅ Готово!"

## 5. Скачивание аудио

1. Вставь ссылку на видео
2. Нажми на **🎵 Аудио (MP3)** в переключателе режимов
3. Появятся кнопки с доступными аудиоформатами:

| Кнопка | Описание |
|--------|----------|
| **128kbps** | Хорошее качество |
| **320kbps** | Лучшее качество |

Слева — битрейт, справа — размер файла и формат (например, "3.3 MB · M4A").

4. Нажми на нужную кнопку и дождись загрузки

## 6. Где находятся скачанные файлы

Все файлы сохраняются в папку **`downloads`** рядом с приложением, сортированные по платформам:

```
downloads/
├── youtube/       # Видео с YouTube
├── instagram/     # Видео с Instagram
├── facebook/      # Видео с Facebook
├── linkedin/      # Видео с LinkedIn
├── twitter/       # Видео с Twitter/X
├── tiktok/        # Видео с TikTok
└── other/         # Видео с других сайтов
```

### Открыть папку

Нажми кнопку **"📁 Загрузки"** внизу окна — папка откроется в файловом менеджере.

### Имена файлов

Формат: `Название видео [ID].расширение`

Пример: `Never Gonna Give You Up [dQw4w9WgXcQ].mp4`

> **Примечание:** Длина имени файла ограничена 120 символами для совместимости с Windows.

## 8. Решение проблем

### Не запускается

| Проблема | Решение |
|----------|---------|
| "Python не найден" | Установи Python 3.7+ с [python.org](https://www.python.org/downloads/) |
| "yt-dlp не найден" | Проверь интернет-соединение, перезапусти приложение |
| Антивирус блокирует | Добавь папку приложения в исключения антивируса |

### Не скачивается

| Проблема | Решение |
|----------|---------|
| "Не удалось проверить ссылку" | Проверь, что ссылка рабочая и видео публичное |
| "Ошибка загрузки" | Посмотри `instrumentarium.log` рядом с .exe |
| Низкое качество | Установи ffmpeg (на Windows устанавливается автоматически) |

### Видео без звука

Если видео скачалось без звука:
1. Убедись, что ffmpeg установлен (на Windows — автоматически)
2. Перезапусти приложение
3. Попробуй снова

### LinkedIn / Приватные страницы

Для скачивания видео с LinkedIn (и других платформ, требующих авторизацию):
1. Установите расширение для браузера **"Get cookies.txt LOCALLY"**
2. Откройте linkedin.com и войдите в аккаунт
3. Нажмите на иконку расширения → **Export**
4. В приложении нажмите **🍪 Cookies** → перетащите файл cookies.txt или вставьте текст
5. Нажмите **Сохранить** → окно закроется автоматически

Теперь приложение может скачивать видео, доступные под вашим аккаунтом.

### Логи

Подробная информация об ошибках записывается в файл **`instrumentarium.log`** рядом с приложением. При обращении за помощью приложи этот файл.
